The Adobe Edge Animate 5.0.1 runtime is provided here for your reference.

In order to substitute this runtime for the minified equivalent you will need to edit the xyz.html file generated by Animate 5.0.1. You should replace edge.5.0.1.min.js with edge.5.0.1.js.
